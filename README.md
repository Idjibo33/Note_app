# Notes

Mini application de prise de note 


## Fonctionnalités 
- Création de note et enregistrement dans la base de donnée local(Titre, texte)
- Modification de note et enregistrement dans la base de donnée local 
- Suppression de note de la base de donnée local
- Choix de thème (sombre/clair)
- Recherche de note 


## Statut de l'application :
***Ceci est un MVP, il va evoluer au fil du temps***
- Front-End (UI/UX) Complet Le design et la mise en page sont finalisés et réactifs pour un MVP.

- Logique d'enregistrement de note complet pour un MVP.

- Déploiement Non applicable, *Projet d'apprentissage, non destiné à la production.*

## Objectif de ce projet :
Ce projet a été développé dans le cadre de mon apprentissage de Flutter et visait principalement à maîtriser les points suivants :

- Logique d'interaction : Utilisation de paramètres entre les pages.

- Manipulation des données : Utilisation du plugin sqflite pour la gestion d'une base de donnée relationnelle pour gérer le stockage local des données. 

-Gestion des états : Utilisation du gestionnaire d'etat providers pour faciliter le partage de données entre les widgets.
